## :runner: Run Locally

Clone the project

```bash
  git clone https://github.com/trstefan/firebase-todo.git
```

Go to the project directory

```bash
  cd firebase-todo
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
